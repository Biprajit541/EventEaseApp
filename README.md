# EventEase

## Project Overview

EventEase is a Blazor web application developed for managing corporate and social events. The application allows users to browse available events, view event details, register for events, and track their registrations through an attendance tracker.

---

## Project Requirements

### ✅ GitHub Repository
A GitHub repository was created for the EventEase project to manage the source code, track development progress, and maintain version control throughout the project.

### ✅ Event Card Component
Microsoft Copilot was used to generate the foundational Event Card component. The component displays the event name, date, and location while using Blazor data binding to dynamically display event information.

### ✅ Routing Functionality
Routing was implemented between the Home, Event List, Event Details, Registration, and Attendance pages. Navigation was tested and improved to ensure users can move smoothly throughout the application, with a custom NotFound page handling invalid routes.

### ✅ Performance Optimization
The application was optimized by adding input validation using Data Annotations, improving routing reliability, organizing business logic into reusable services, and reducing duplicated code for better maintainability.

### ✅ Advanced Features
Advanced functionality includes a Registration Form with validation, user session management using Blazor services, and an Attendance Tracker that records registered events and displays their details.
